Event Management System

Project Description

This is a full-stack Event Management System developed using MERN stack. It allows Admin to manage events and Users to register for events.

 Features

Admin

- Add Event
- Update Event
- Delete Event
- Search Event
- View Participants

User

- Register/Login
- View Events
- Search Events
- Register for Event

 Technologies Used

- React.js
- Node.js
- Express.js
- MongoDB

 Folder Structure

frontend → React app
backend → Node + Express API

 How to Run

Backend

cd backend
npm install
npm run dev

Frontend

cd frontend
npm install
npm start
